# EDI Workflow Demo

An end-to-end healthcare EDI claims processing demo built with:

- **Java** — EDI parsing, claims processing, REST API
- **PostgreSQL** — Claims, payments, and validation log storage
- **Python** — Duplicate detection, metrics, analytics
- **Vue** — Dashboard, charts, visualizations
